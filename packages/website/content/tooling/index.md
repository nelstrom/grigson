# Tooling

Tools for working with `.chart` files from the command line and in the browser.

- [CLI](/tooling/cli/) — normalise, transpose, validate, and render `.chart` files from the terminal
- [Playground](/tooling/playground/) — live editor with syntax highlighting, normalisation, and URL sharing
- [AST Explorer](/tooling/ast-explorer/) — inspect the parsed AST of any chart, with source-range highlighting
