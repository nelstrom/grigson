# Usage

This section covers embedding grigson charts on a webpage.

- [Getting started](/usage/getting-started/) — load the browser bundle and render your first chart
- [Custom elements](/usage/custom-elements/) — `<grigson-chart>` and `<grigson-html-renderer>` for declarative HTML embedding
- [HTML renderer](/usage/html-renderer/) — grid layout, HTML structure, parts reference, and configuration
- [Build-time rendering](/usage/build-time/) — pre-render charts into Declarative Shadow DOM at build time for zero-JS display
- [Accessibility](/usage/accessibility/) — ARIA labels, spoken presets, and screen-reader behaviour
