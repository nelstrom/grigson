# Grigson VS Code Extension

Provides syntax highlighting for Grigson `.chart` files.

## Features

- Syntax highlighting for front matter, barlines, and chords.
- Highlighting for all supported qualities: major, minor (`m`), dominant seventh (`7`), and half-diminished (`m7b5`).
- Visual distinction between chord roots and accidentals.

## Development

To test the extension:
1. Open this directory in VS Code.
2. Press `F5` to launch an Extension Development Host.
3. Open a `.chart` file (see `packages/grigson/documentation/examples/syntax-test.chart`).
