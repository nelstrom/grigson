# Normalisation Demo

This page demonstrates the automatic normalisation of a chord chart. 
The input below is "badly" spelled (e.g. using A# in F major) and has inconsistent spacing.
The output is generated by `grigson` in your browser.

## Input

<pre id="input">
---
title: Demo Song
key: F
---

| F | A# | F | C7 |
|F|Bb|C|F|
</pre>

## Output

<pre id="output">Loading...</pre>

<script>
  window.addEventListener('DOMContentLoaded', () => {
    const inputElement = document.getElementById('input');
    const outputElement = document.getElementById('output');
    const input = inputElement.textContent.trim();
    
    try {
      const song = grigson.parseSong(input);
      const normalised = grigson.normaliseSong(song);
      const renderer = new grigson.TextRenderer();
      outputElement.textContent = renderer.render(normalised);
    } catch (e) {
      outputElement.textContent = 'Error: ' + e.message;
    }
  });
</script>
